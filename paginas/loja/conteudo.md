# Loja Virtual / Orcamento - CAPOL

**URL:** https://capol.com.br/carrinho (Orcamento/Carrinho)
**URL de Produtos:** https://capol.com.br/produtos

*Nota: O site capol.com.br nao possui uma pagina "/loja" dedicada. O sistema funciona como catalogo de produtos com funcao de orcamento (carrinho).*

---

## Como Funciona

O site da CAPOL funciona como um catalogo online onde os cooperados podem:
1. Navegar pelas categorias de produtos
2. Visualizar detalhes dos produtos
3. Adicionar produtos ao carrinho ("Adicionar")
4. Ver detalhes do produto ("Ver Produto")
5. Solicitar orcamento atraves do carrinho
